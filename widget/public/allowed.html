Bloom Florista

This page's origin (http://localhost:3000) is in the demo widget's allowed_origins, so the loader will exchange a token and inject the chat iframe in the bottom-right corner.

Pass the widget UUID in the URL: ?widget_id=<UUID>. Get the seeded Bloom widget id with: 

docker compose exec -T db psql -U concierge -d concierge -tAc \
  "SELECT id FROM widgets WHERE tenant_id='aaaaaaaa-aaaa-aaaa-aaaa-aaaaaaaaaaaa' LIMIT 1;"
